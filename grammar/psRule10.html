Части речи

Предлог


Предлог — это служебная часть речи, которая стоит перед словами, изменяемыми по падежам, и служит для связи слов в предложении. Предлоги не изменяются и не являются членами предложения.


по происхождению
непроизводные производные
в, без, до, из, к, на, по, о, от, перед, при, через, у, за, над, об, под, про, для наречные отымённые глагольные
около, возле насчёт, ввиду, в продолжение несмотря на, благодаря, спустя


по составу
простые из одного слова: с, около, вблизи
составные из нескольких слов: в течение, в связи
сложные из двух простых, пишутся через дефис: из-под, из-за
