Цель проекта


Цель проекта заключается в том, чтобы собрать в одном месте всю информацию, необходимую для подготовки к экзамену.

Задачи:

  1. Собрать и упорядочить грамматические правила. Сделать их удобными к запоминанию.
  2. Предоставить схему решения для каждого задания первой части. Сделать удобным просмотр правил в схеме заданий.
  3. Сформулировать пошаговый план написания сочинения, в котором есть список тем, шаблоны и примеры.
  4. Показать, как оцениваются работы. Предотвратить ошибки на экзамене.
  5. Сформировать представление о виде экзамена при помощи вариантов ЕГЭ.
  6. Помочь с подготовкой со всех сторон, чтобы ученик избежал стресса и ошибок на экзамене.
